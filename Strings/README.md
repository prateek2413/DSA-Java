# Strings

This folder contains my Java solutions to **String** problems from coding platforms such as LeetCode. The focus is on building a strong understanding of string manipulation techniques, common interview patterns, and efficient algorithms.

## Topics Covered

* String Traversal
* Character Frequency
* String Manipulation
* HashMap with Strings
* Two Pointers
* Sliding Window
* Palindrome Problems
* Anagrams
* Prefix & Suffix
* Pattern Matching

## Language

* Java

## Goal

To develop strong problem-solving skills by mastering string-based algorithms and interview techniques commonly used in software engineering interviews.
